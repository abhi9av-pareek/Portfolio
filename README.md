## Setup Instructions

1=>Clone the repository
```bash
git clone https://github.com/abhi9av-pareek/portfolio.git
cd portfolio
```
2=>Install dependencies
```bash
npm install
```
3=>Run the project
```bash
npm start
```
Then visit => http://localhost:3000
